## ¿Qué es CompuClub?

CompuClub es un emprendimiento fundado por **Vicente Cayetano** y **Gabriela López** que nació en Zacoalco de Torres, Jalisco, con una misión clara: cerrar la brecha digital en su comunidad.

Creemos que la tecnología no debería tener fronteras. Que cada persona en nuestra comunidad merece aprender, crear y conectar con el mundo digital, sin importar su edad ni experiencia previa.

Ofrecemos cursos y formación tecnológica accesible para todas las edades. **CompuClub** es nuestra forma de devolver algo a nuestra comunidad.

---

## 🚀 Stack Tecnológico

- **Next.js 16.2.0** — Framework React con renderizado híbrido y App Router
- **React 19** — Biblioteca de interfaces de usuario
- **Tailwind CSS 4.2.0** — Framework de utilidades CSS
- **Framer Motion 12.38.0** — Animaciones declarativas
- **TypeScript 5.7.3** — Tipado estático opcional
- **Radix UI** — Componentes primitivos accesibles
- **Lucide React** — Iconos
- **Vercel Analytics** — Medición de analytics
- **Zod** — Validación de esquemas
- **React Hook Form** — Formularios performantes

## ⚙️ Instalación y uso local

### Requisitos previos

- Node.js 18.x o superior
- npm (incluido con Node.js)

### Pasos

```bash
# Clonar el repositorio
git clone https://github.com/Vicente28CF/landing-page-compuclub

# Entrar al proyecto
cd landing-page-compuclub

# Instalar dependencias
npm install

# Iniciar en desarrollo
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.
